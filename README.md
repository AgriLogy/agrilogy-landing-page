# Agrilogy Landing Page

Landing page for Agrilogy — smart agriculture solutions (IoT, irrigation, precision farming).

## Tech

- **Next.js 15** (App Router)
- **Chakra UI**
- **Swiper** (hero & partners carousels)
- **EmailJS** (contact form)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project structure

- `src/app/page.tsx` — main landing page
- `src/app/components/ContactSection.tsx` — contact form & info
- `public/img` — images
- `public/partners` — partner logos
